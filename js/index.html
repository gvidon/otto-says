Строки

Строки конкатенируются плюсом либо методом .join(). Вокруг каждого плюса ставится по одному пробелу.

Везде для строк используются одинарные кавычки.

Длина строки кода

Ограничена 80-ю символами. В исключительных случаях, например чтобы не разбивать булево выражение расширяется до 132-х.

Комментарии

Все комментарии пишутся с большой буквы, с одним пробелом после //. Блок/строка с комментарием отделяется от следующего блока/строки, к которому не относится комментарий пустой строкой.

Определение переменных

Блок var всегда обязан быть первым в опредлении функции. Все переменные должны быть определены до первого использования.

Каждая переменная, а также var должны быть на отдельной строке и отсортированы по алфавиту (пока это возможно). Каждое определение переменной выделяется отступом:

var
	arr = [1, 2, 3],
	c = 300,
	b = c + 1;

Определение единстввенной переменной происходит на одной строке с var:

var variable = 3;

В функциях итераторах переменные стоит называть заглавной литерой, которая является первой в названии сущности переменной. Например:

persons.map(function(P) { return P.getName() })

Определение функций

Пробел между function и ([param, [params, ...]]) у анонимных функций ставить не нужно. Между выражением однострочной функции и фигурными скобками внутри должно быть по одному пробелу.

Вложенные функции определяются сразу после блока var.

Блоки кода (выражения)

Блок в одну строку не должен выделяться фигурными скобками.

Если выражение блока и оператор if/else/for/while умещается в 132 символа, то весь блок стоит оформить на одной строке.

Содержимое скобок () не отделяется от них пробелом

Вертикальное выравнивание

К списку переменных или содержимому массива/объекта применяется вертикальное выравнивание значений, если в списке больше 3-х элементов. Например:

var fruits = {
	apple : 'green',
	banana: 'yellow',
	orange: 'orange',
	peach : 'red'
};
