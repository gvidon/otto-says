Перенос строк

Не стоит применять никакие правила логической/стилистической группировки при добавлении новой строки в js. Перенос строки нужно использовать только перед/после многострочными блоками и перед/после участка комментированного кода.

someCall(param);
object.method(objectParam);
anotherObject(anotherParam);
tryThisone(param).andChainCall(here);
return this;
 
# This is comment
commented.call();
thisCall.relatesToCommentAbove();

object.method(objectParam);
anotherObject(anotherParam);
 
if(1 > 0)
  call();
  
callAfter.ifCondition();
